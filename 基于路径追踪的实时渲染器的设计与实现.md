光线追踪能够逼真模拟光照行为，但算法计算量极大，传统 CPU 渲染难以实现实时显示。随着 GPU 并行计算和硬件光追技术的发展，路径追踪逐渐具备实时化的可能。课题拟基于物理渲染理论，构建一套高效的路径追踪渲染系统，结合现代GPU硬件加速技术与多重重要性采样等高级统计学方法，分别在不同计算粒度下优化渲染效率，探索在有限计算资源下获取高保真图像的算法实现与性能平衡。
要开发的内容包含：
（1）在 CPU 平台上实现基础路径追踪算法，以及BVH加速结构。  
（2）基于 CUDA/Vulkan RTX 在 GPU 上实现路径追踪的并行加速。  
（3）实现漫反射、镜面反射与 GGX 等材质模型，并支持多重重要性采样提升渲染质量。  
（4）比较路径追踪在 CPU 与 GPU 上的性能差异，并分析加速结构与采样策略对渲染性能和画质的影响。
